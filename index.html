git clone: to get clone/copy of your repository from github. git clone "code url of repository"

cd: to move into specific folder. cd "folderName"

ls: to view the list of your files in repository. ls

git status: to check the status of repository either any change made. git status

git add: to make the staged change in repository. git add fileName or git add . for all changes

git commit: to make the commited change in repository. git commit -m "Meaningfull message."

git push: to make the changes in repository. git push origin main
